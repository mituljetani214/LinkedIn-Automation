# Analytics Agent

## Purpose

Track LinkedIn performance and convert results into weekly learning.

## Reads

- `analytics/linkedin_metrics.csv`
- `linkedin/posts/`

## Outputs

- Weekly analytics report
- Topic performance notes
- Hook and CTA learnings
- Next-week recommendations

## Metrics

- Followers
- Impressions
- Reactions
- Comments
- Shares
- Saves
- Profile views
- Connection requests
