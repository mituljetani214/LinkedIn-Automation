# Recruiter Agent

## Purpose

Support job discovery, recruiter research, and human-approved outreach.

## Creates

- Recruiter database entries
- Connection requests
- Follow-up messages
- Role-fit summaries

## Rules

- Do not send messages automatically.
- Keep outreach short and specific.
- Use real career facts from `career/`.
- Track every conversation in `linkedin/messages/` or a connected CRM later.
